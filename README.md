## Student Grade Calculator (C++)

This repository contains a console-based Student Grade Calculator developed in C++  
as part of Task 4 assigned by Interncred during the internship.

The program calculates total marks, percentage, grade, and pass/fail status for  
multiple students across multiple subjects.

## Features
- Input marks for 5 subjects
- Support for multiple students
- Calculates:
  - Total marks
  - Percentage
  - Grade (A, B, C, D, F)
- Displays Pass/Fail status
- Automatically fails student if any subject mark is below passing criteria

## Grading Logic
- A: Percentage ≥ 90
- B: Percentage ≥ 75
- C: Percentage ≥ 60
- D: Percentage ≥ 40
- F: Percentage < 40

## Concepts Covered
- Arrays
- Loops
- Conditional statements
- Input/output handling
- Basic grading logic
